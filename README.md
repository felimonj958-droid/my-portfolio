# Felimon Jimenez - Portfolio

This is my personal portfolio website showcasing my data science experience, skills, and projects.

## Live Site

This site is deployed using GitHub Pages.

## GitHub Pages Setup

To configure GitHub Pages for this repository:

1. Go to your repository on GitHub: `https://github.com/felimonj958-droid/my-portfolio`
2. Click on **Settings** (gear icon in the top menu)
3. Scroll down and click on **Pages** in the left sidebar
4. Under **Source**, select:
   - Branch: `main`
   - Folder: `/ (root)`
5. Click **Save**
6. Your site will be published at: `https://felimonj958-droid.github.io/my-portfolio/`

## Local Development

To view the site locally, simply open `index.html` in a web browser.
